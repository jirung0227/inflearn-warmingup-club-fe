### 인프런 워밍업 클럽 스터디와 함께해요! .feat John Ahn님

실무에서 기초가 중요하다는 건 아무리 강조해도 지나치지 않죠. 인프런 워밍업 클럽 스터디에 참여하고, 취업 준비와 실무에 필요한 기초를 탄탄히 쌓아보세요!
